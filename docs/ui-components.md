# UI Components

Custom component library located in `src/shared/ui/`. All components use CSS Modules (`.module.scss`).

## Layout

| Component | Description |
|---|---|
| `Stack` | Flex container with HStack / VStack variants |
| `Card` | Base card wrapper |
| `CardsGrid` | Grid layout for cards |
| `Modal` | Modal dialog |
| `Drawer` | Slide-out panel (mobile navigation) |
| `Overlay` | Background dimming overlay |
| `Portal` | React Portal for rendering outside the DOM tree |
| `Accordion` | Collapsible block |
| `FormSection` | Form section with heading |

## Forms

| Component | Description |
|---|---|
| `Button` | Button with style variants |
| `Input` | Text input field |
| `Textarea` | Multi-line text input |
| `CheckBox` | Checkbox |
| `RadioButton` | Radio button |
| `Select` | Custom select (react-select) |
| `CounterInput` | Numeric input with +/- buttons |
| `RangeInput` | Range slider (for price filter) |
| `Tag` | Tag button (for filters) |
| `Toggler` | Toggle switch |
| `Form` | Form wrapper |
| `Search` | Search input with icon |

## Navigation

| Component | Description |
|---|---|
| `AppLink` | Typed wrapper around React Router Link |
| `DecoratedLink` | Styled link with icon |
| `GoHomeButton` | Go-to-home button |
| `Breadcrumbs` | Breadcrumbs with automatic path building |
| `Tabs` / `Tab` | Tab navigation |

## Feedback

| Component | Description |
|---|---|

| `OnlineStatus` | Online/offline indicator |
| `Loaders` | Loaders / spinners |
| `Skeleton` | Content loading skeleton |

## Media / Sliders
